How to Use Lab Manager Config

Step 1: Upload Your Files
  • Upload your group_config.xml and thumbnail_settings.xml files
  • Optionally, enter a list of active IP addresses to pre-select systems
  • Check the "Use IP Address List" box to enable IP-based selection
Step 2: Edit Your Files
  • Use checkboxes to select which systems to keep
  • Systems are organized by access code (00, 01, 02)
  • Your selections are saved as you move between pages
  • If using IP list:
    • Systems with matching IPs will be pre-checked
    • Systems with non-matching IPs will be unchecked
Step 3: Generate New Files
  • Click "Generate New Configuration" to create updated files
  • Generated files are saved in the New Configurations page
  • Download your files from the New Configurations page when ready
Additional Features
  • All uploaded files are automatically archived
  • Access the Archive page to view or download previous uploads
  • Use the New Configurations page to manage generated files
  • Delete individual files or all files from either page
